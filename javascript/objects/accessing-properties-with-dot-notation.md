# Accessing object properties with dot notation

There are two ways to access object properties, to get the value of a property from an object.

The most common way is called dot notation using a `.`.

If we had this object:

```js
const product = {
    id: 9264,
    name: "Lovely product",
    price: 9.99
}
```

we could access its `id` property and assign it to a variable like this:

```js
const producId = product.id;
```

We could access its `price` property and assign it to a variable like this:

```js
const producPrice = product.price;
```
